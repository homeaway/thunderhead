Thunderhead

Thunderhead - An AWS Cloudsearch client for updating and querying your search domains written in Java

View GitHub project »

Amazon Documentation

Amazon Web Services have built out a REST API for communicating with Cloudsearch. Thunderhead is a client for sending HTTP requests to this REST API for querying and updating your search domain. Amazon has documented this interaction with Cloudsearch.
